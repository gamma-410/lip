Welcome to lip!

lip は 最小限の構成 で 最低限必要 な UI・UX を構築できることを目標に開発されました。 モバイルファーストに対応できるサイトを構築しましょう。

lip was developed with the goal of allowing you to build a minimal UI / UX with a minimal configuration. First, let's build a site that supports mobile.


Start lip

まずは github/gamma410 から lip をダウンロードしましょう。

その後 lipディレクトリ を自分のプロジェクトのルート配下に設置します。

早速プロジェクトに lip.css, lip.js を追加しましょう。


First, download the lip from github / gamma410.

Then place the lip directory under the root of your project.

Add lip.css and lip.js to your project right away.


        <!doctype html>
        <html lang="">
        <head>
            <!-- required meta tags -->
            <meta charset="utf-8">
            <meta name="viewport" content="width=device-width, initial-scale=1">
            

            <!-- lip CSS -->
            <link href="lip/lip.css" rel="stylesheet">
            

            <title>example - Hello, world!</title>
        </head>
        <body>
            <h1>Hello, world!</h1>
            

            <!-- lip JavaScript -->
            <script src="lip/lip.js"></script>
        </body>
        </html>

CSS


        <!-- lip CSS -->
        <link href="lip/lip.css" rel="stylesheet">

JavaScript


        <!-- lip JavaScript -->
        <script src="lip/lip.js"></script>
