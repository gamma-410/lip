Welcome to lip!

lip は 最小限の構成 で 最低限必要 な UI・UX を構築できることを目標に開発されました。 モバイルファーストに対応できるサイトを構築しましょう。

lip was developed with the goal of allowing you to build a minimal UI / UX with a minimal configuration. First, let's build a site that supports mobile.

Start lip

まずは github/gamma410 から lip をダウンロードしましょう。

その後 lipディレクトリ を自分のプロジェクトのルート配下に設置します。

早速プロジェクトにlip.css, lip.jsを追加しましょう。
